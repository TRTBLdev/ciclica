# CÍCLICA 🌿

> Bóveda Soberana de Vida y Energía • Local-First \& Open Source.

🌐 **Acceder a la aplicación web**: [ciclica.trtbl.dev](https://ciclica.trtbl.dev)  
🔬 **Laboratorio de ideas**: [The Right to be Lazy](https://www.trtbl.dev)

CÍCLICA es un gestor de energía ejecutiva y ritmo de vida diseñado bajo principios de soberanía de datos y privacidad crítica. La aplicación calibra y sintoniza tu agenda diaria y tus arquetipos cognitivos de enfoque en base a tu biología natural (ciclo menstrual o lunar) de forma 100% desconectada y física en tu dispositivo.

\---

## 💡 Filosofía y Características

* **🔒 Privacidad Crítica \& Local-First**: Ningún dato sale de tu dispositivo. Toda tu información (pulsos, bitácoras, hábitos y configuración) reside físicamente en el almacenamiento local de tu navegador (`localStorage`).
* **🩸 Sintonía Menstrual Adaptativa**: Calibra dinámicamente tu Barra de Energía Ejecutiva y tus arquetipos cognitivos de enfoque (Oro/Dinámica, Coral/Expresiva, Turquesa/Creativa, Azul Lino/Reflexiva) según las fases de tu ciclo biológico, con un algoritmo adaptativo que auto-calibra tus duraciones medias basándose en tu bitácora de sangrado.
* **🌙 Ritmología Lunar**: Motor de sincronización sinódico lunar predictivo alternativo basado en la fecha real del sistema, ideal para periodos de gestación, menopausia o desconexión hormonal.
* **💼 Planificación Orgánica**: Gestión de tareas por bloques horarios, seguimiento de hábitos y botes de energía ejecutiva en lugar de metas de rendimiento lineales corporativas.
* **📦 Portabilidad Absoluta**: Importa y exporta tu bóveda en formato `.json` en cualquier momento con un solo clic.

## 🛠️ Tecnologías

* **Core**: React 19 (TypeScript)
* **Bundler**: Vite 6
* **Estilos**: CSS nativo y TailwindCSS para transiciones y micro-animaciones fluidas
* **Iconos**: Lucide React
* **Animaciones**: Motion

## 🚀 Ejecución Local

### Prerrequisitos

Tener instalado [Node.js](https://nodejs.org/).

1. **Instalar dependencias**:

&#x20;   ```bash
    npm install
    ```

2. **Iniciar servidor de desarrollo**:

&#x20;   ```bash
    npm run dev
    ```

   La aplicación se abrirá en `http://localhost:3000`.

3. **Compilar para producción**:

   &#x20;   ```bash
    npm run build
    ```

   Los archivos estáticos listos para desplegar (en Vercel, Netlify o cualquier servidor) se generarán en la carpeta `dist`.
